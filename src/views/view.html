Directions

  • How to add our privacy policy display to your project

  • 1: Run our .js library locally

    Copy and paste the JSON and javascript into your own .js file. Make sure to link to the .js file in the body of your HTML.

    <script src="path/to/pps.js"></script>

    If using an MVC (e.g. Angularjs), add the JSON / js into the apporpriate controller.

    Copied!

    var mpnData = {{view.jsonPreview}}

  • 2: Link to our CSS

    Copy and paste this link to our CSS library into the

    <head>of your project.

    <link rel="stylesheet" href="https://videntity.github.io/pps-display/lib/styles/main.min.css" type="text/css">

  • 3: Create a container element for the display

    Please make sure to use our className:

    <div class="policyDisplay"></div>

    This container can be nested within your own container with a minimum width of 400px.
