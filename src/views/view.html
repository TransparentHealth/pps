Directions

  • How to add our privacy policy display to your project

  • 1: Copy the Json object into your project.

    Copy and paste the JSON into your own .js file. Make sure to link to it in the body of your html.

    <script src="path/to/mpnJson.js"></script>

    Copied!

    var mpnData = {{view.jsonPreview}}

  • 2: Link to, or download our javascript library.

    Link to our library in the body of your html, BELLOW your jquery link and the JSON. Please note: the privacy notice is created entirely with jquery.

    <script src="https://videntity.github.io/pps-display/lib/js/app.min.js"></script>

    OR download our library. If using an MVC (like Angular.js), put the JSON and javascript in the appropriate controller/service.

  • 3: Link to our CSS

    Copy and paste this link to our CSS library into the

    <head>of your project.

    <link rel="stylesheet" href="https://videntity.github.io/pps-display/lib/styles/main.min.css" type="text/css">

  • 4: Create a container element for the display

    Please make sure to use our className: "policyDisplay"

    <div class="policyDisplay"></div>

    This container can be nested within your own container element with a minimum width of 375px.
